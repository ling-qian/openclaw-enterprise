# Skills 索引

企业版预置技能列表。所有技能也可在 [ClawHub](https://clawhub.ai) 独立安装。

## 安全类

| 技能 | 说明 | ClawHub Slug |
|------|------|--------------|
| safety-audit | 安全审计检查清单 | `safety-audit` |
| api-security | API 安全测试 | `api-security` |
| threat-hunting | 威胁狩猎 | `threat-hunting` |
| cloud-security | 云安全审计（AWS/GCP/Azure） | `cloud-security` |
| incident-response | 安全事件响应 | `incident-response` |

## 开发类

| 技能 | 说明 | ClawHub Slug |
|------|------|--------------|
| wechat-lead-generation | 微信线索抓取与分析 | `wechat-lead-generation` |

## 使用方法

```bash
# 安装预置技能
openclaw skills install <slug>

# 或从 ClawHub 直接安装
clawhub install <slug>

# 搜索更多技能
openclaw skills search "keyword"
```
