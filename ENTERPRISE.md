# OpenClaw Enterprise 企业版

OpenClaw Enterprise 是针对中大型组织的 AI Agent 管理与编排平台，基于开源 Hermes Agent 构建，提供企业级安全、合规、运维和支持服务。

**适合对象**：需要将 AI Agent 应用于安全运营、合规审计、渗透测试、威胁狩猎、云安全等场景的企业客户。

---

## 核心价值

- **统一管理**：集中管理数百个 AI Agent，统一技能库、模型路由、任务调度
- **安全合规**：私有部署、数据驻留、审计日志、技能白名单、SOC2 准备
- **可观测性**：任务追踪、性能分析、成本可视、告警集成
- **开箱即用**：预集成 30+ 安全技能（渗透测试、云安全、威胁狩猎等）
- **专家支持**：24x5 工程师支持 + 定制开发服务

## 安全与合规

### 数据保护
- 数据传输：TLS 1.3 强制
- 静态加密：AES-256（AWS KMS 或客户自带 HSM）
- 无第三方日志：可关闭所有外部 API 调用
- 数据删除：支持 GDPR 删除请求，7 天内完成

### 认证与授权
- SAML 2.0 / OIDC 单点登录（Okta, Azure AD, Google Workspace, Keycloak…）
- 基于角色的访问控制（RBAC）：
  - Viewer（只读）
  - Operator（执行任务）
  - Admin（配置管理）
  - Security（审计日志）
- 技能执行白名单：仅允许认证技能运行

### 审计与追踪
- 结构化审计日志（JSON）包含：
  - Who（执行人）
  - What（agent + skill + 输入摘要）
  - When（UTC 时间戳）
  - Result（输出哈希、退出码）
- 日志保留：默认 1 年，可扩展到 7 年
- 导出格式：Syslog、HTTP Webhook、S3 桶

### 合规准备
- **SOC 2 Type II**：审计就绪文档包（含控制矩阵、证据收集脚本）
- **ISO 27001**：信息安全政策模板（提供给你法务团队）
- **NIST 800-53**：映射到 OpenClaw 的配置控制项

## 技术架构（托管版）

```
┌─────────────────┐
│   Client UI     │ ← React SPA (CloudFront)
└────────┬────────┘
         │ HTTPS
┌────────▼────────┐
│ Load Balancer   │ ← ALB/Nginx
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
┌───▼───┐ ┌──▼───┐
│ Agent │ │ Agent│  ← 横向扩展 (ECS/EKS)
│  #1   │ │  #N  │
└───────┘ └──────┘
    │         │
    └────┬────┘
         │
┌────────▼────────┐
│   PostgreSQL    │ ← RDS (加密)
└─────────────────┘
```

**关键组件**：
- API Gateway (FastAPI) + PostgREST 风格设计
- Task Queue (Redis / SQS)
- Model Router（成本优化 + 限流）
- Skills Registry（S3 存储，版本化）
- Audit Sink（Kafka → S3 + Elastic）
