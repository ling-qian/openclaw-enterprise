```

**关键话术**：

> "传统方式下，一个安全工程师需要 2-3 天完成这些检查。

### Step 4: 展示结果报告（4 分钟）

```
报告生成完毕。这是自动输出的审计报告：

Executive Summary:
- 扫描资产：47 个 S3 桶 + 3 个 K8s 集群
- 发现问题：22 个

（逐个展示 Critical 发现）

⛔ Critical #1: S3 桶 "customer-data-prod" 公开访问
   影响：任何人可读取客户数据
   修复：aws s3api put-bucket-acl --bucket customer-data-prod ...
   一键修复命令已生成

   影响：任何绑定此角色的 Pod 可执行所有操作

**关键话术**：

## 客户常见问题（FAQ）

**Q: 支持哪些云平台？**
> A: AWS、Azure、GCP 都支持。Kubernetes 和 Terraform 也覆盖。

**Q: 数据安全吗？**
> A: 支持私有部署，你的数据不出你的网络。也提供 SaaS 版本，数据加密传输和存储。

**Q: 和商业扫描器（Prisma Cloud、Wiz）有什么区别？**
> A: 他们是资产发现平台，我们是执行层。可以配合使用 — 他们发现资产，我们自动修复。

**Q: 你们团队多大？**
> A: 我们是开源项目起步，已经有 30+ 安全技能在 ClawHub 上发布，GitHub 有 2k+ stars。

**Q: 能定制吗？**
> A: 可以。$200/小时定制开发，或者按项目报价。年付客户包含 20 小时/月。
